# zzc-tool
## 基于commander的CLI工具
---

## Feature

- 向任意账号发送反馈消息(默认本身) zzc cs [-u xxxx]  xxxxx

- 单词翻译 zzc fy xxx

- 图片压缩 zzc ys

- 查看本机ip并复制到剪贴板 zzc ip 

- 获取字符unicode符号 zzc unicode xxx

- 添加ssh key zzc ssh

## Install
### SETP 1
`yarn add global zzc`
or
`npm i -g zzc `
### SETP 2
修改config/index.js填写自己配置
